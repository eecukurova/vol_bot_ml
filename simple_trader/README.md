# Simple Trader Project

## Project Structure

- **eigen/** - EIGEN/USDT Trading System
- **bist/** - BIST Signal Generator  
- **nasdaq/** - NASDAQ Signal Generator
- **common/** - Common Files & Tests
- **logs/** - Log Files
- **state/** - State Files

## Services

- **eigen-trader.service** - EIGEN/USDT Auto Trading
- **bist-signal-generator.service** - BIST Signal Generator
- **nasdaq-signal-generator.service** - NASDAQ Signal Generator

## Status

- **EIGEN**: Active trading with position management
- **BIST**: Signal generation for Turkish stocks
- **NASDAQ**: Signal generation for US tech stocks under $1
